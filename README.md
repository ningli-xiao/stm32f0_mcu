2.长时间无操作后模组关机
原因：不外接电池电流不稳定
解决：插入内电池
3.首次登录后发送pub包失败
原因：可能是有波动导致串口接受到不正常的上传数据
解决：加入过滤后无问题
4.电池没电时频繁关机重启
原因：模组无法开机时某些函数运行时间过长
解决：拆分，缩短运行时间小于10S
